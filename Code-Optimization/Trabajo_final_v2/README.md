# Trabajo_final_v2

Refactor de `Trabajo_final` orientado a mejorar orden, legibilidad y trazabilidad sin cambiar deliberadamente el comportamiento observable general del sistema.

## Estructura

- `app/`: código de la aplicación.
- `data/`: archivos JSON/TXT usados por la versión refactorizada.
- `docs/`: documentación técnica y guía de pruebas manuales.
- `prds/`: documento de planificación funcional.
- `tests/`: pruebas automatizadas sobre piezas estables.

## Ejecución

```bash
python3 main.py
```
